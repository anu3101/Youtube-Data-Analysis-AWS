# AWS Data Analysis-Youtube Data
# <h2> Introduction
This project involves building a data pipeline using various AWS services to collect, store, process, and visualize YouTube data. The data is initially stored in CSV and JSON formats, then processed and transformed into Parquet format for efficient querying and analysis. Finally, the data is visualized using AWS QuickSight to provide actionable insights.
# <h2> AWS Services:

AWS S3: For storing raw and processed data.<br>
AWS Glue: For crawling the data and creating a data catalog.<br>
AWS Lambda: For performing ETL (Extract, Transform, Load) operations.<br>
AWS Athena: For running SQL queries on the processed data.<br>
AWS QuickSight: For creating dashboards and visualizations.<br>
# <h2>Conclusion
This project demonstrates the use of AWS services to build a robust data pipeline for collecting, processing, and visualizing YouTube data. By leveraging AWS S3, Glue, Lambda, Athena, and QuickSight, we can efficiently manage and analyze large datasets to gain valuable insights.
